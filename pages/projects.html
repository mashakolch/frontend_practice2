Бит для Моргенштерна
Описание проекта

Душераздирающий бит Моргенштерна для подростков 12+ с современным звучанием и крутым продакшеном. Проект включает в себя создание веб-приложения для управления звуковыми дорожками и интеграцию с SoundCloud API.

Функциональность
  • Редактор аудио дорожек в реальном времени
  • Интеграция с SoundCloud для публикации
  • Система эффектов и фильтров
  • Адаптивный дизайн для мобильных устройств
Технологии
HTML/CSS Bootstrap JavaScript SoundCloud API
